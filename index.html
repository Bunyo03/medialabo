プロジェクト課題について

プロジェクト課題かく

数当てゲーム

数当てゲームとは、プレイヤーが1から10までの数字の中からコンピュータがランダムに選んだ正解の数字を当てるゲームです。プレイヤーは数字を入力し、正解と一致するまで繰り返し挑戦します。入力した数字が正解より大きいか小さいかがヒントとして表示されるため、それを参考に次の予想を立てます。シンプルながらも論理的思考力や推測力が試される楽しいゲームです。
テキストボックスに整数を指定して回答ボタンを押すと，正解判定の結果が表示されます 回答回数（予想回数）がカウントされ，ページに表示されます 正解すると正解のメッセージが表示されます 3回まちがうと答えが表示されます 4回以上回答すると，ゲームが終わっているというメッセージが表示されます デバッグ用の答えがコンソールにいつも出力されています 数当てのページ

各授業回の学習事項

第1回 WWW と HTML

今回学習したことは、ウェブサイトの編集の仕方、また、そこで扱う様々な要素や、リンクの埋め込み方、画像の貼り方など多くの使い方を学んだ。このことから、今回自身で活用して、ウェブサイトを試行錯誤して課題の通りに作ることができるようになった。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回の授業では文字の色を変えたり、背景を変えることを学んだ。また、シャドウエフェクトも身につけた。また、背景エフェクトで箱を作りその中に文字を入れ、Webサイトをわかりやすく見ることができる用になった。今回の課題提出でもこれらのことを活かして提出する。改行はめんどくさいのでしないから、頑張って見て下さい。

演習の成果:演習2-3:browser

第3回 JavaScript の基本

今回の授業ではJavaScriptをはじめから学び、そのコードの書き方や、難しさを学んだ。
また、今回JavaScriptでコンソールで確認しながらプログラムを書くということをし、それにより、ミスを減らすことができた。
さらに、数当てゲームを作り応用も作ることができた。

演習の成果:演習3-6

第4回 DOM と JavaScript

今回の課題では、JavaScriptで数当てゲームを作る中で、変数のスコープやイベント処理の重要性を学びました。特に、関数の外と中で変数の値がどう扱われるか、またHTML要素の値を正しく取得・表示する方法について理解を深めました。また、応用で学んだことを再度確認することでより理解を深めることができた。

演習の成果:演習4-10
